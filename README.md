### Ejecucion del proyecto

1. Instalacion de dependencias

```
pnpm install
```

2. Ejecucion global frontend y backend

```
pnpm dev
```

Posibilidad de ejecutar frontend o backend por separado

```
pnpm dev:backend
pnpm dev:frontend
```
